>    - `filter_opacity_min`: remove rows below linear alpha threshold
>    - `filter_box` / `filter_sphere`: remove rows outside user spatial masks
>    - `filter_floaters_by_voxel_contribution`: after detailed voxelization, remove splats that contribute to no occupied voxel
>    - `filter-cluster`: before detailed voxelization, run coarse voxelization at large resolution, build connected components with BFS/flood-fill on coarse occupancy grid, keep only component containing `seed_pos`, remove splats outside that component
>    Verify synthetic scene with main room plus distant noisy island keeps only seed component; verify floater filter remains separate and does not replace `filter-cluster`.
> 12. Add CPU voxel baseline with 4x4x4 block alignment. Input Gaussians evaluate opacity at voxel centers against `opacity_threshold`. Verify floor/cube occupancy.
> 13. Add `wgpu` voxelization behind same trait. Verify CPU/GPU mask parity on deterministic fixtures.
> 14. Add voxel fill + carve pipeline:
>    - Input: binary solid occupancy grid from step 12/13, already aligned and thresholded.
>    - Fill mode `exterior-fill`: dilate solid grid by `exterior_fill_dilation` in world units, flood-fill empty space from boundary, skip fill if flood reaches `seed_pos` because volume is not closed.
>    - Fill mode `floor-fill`: for each XZ column, fill from bottom until first solid voxel; use for outdoor/terrain scenes, not with `exterior-fill`.
>    - Carve: flood-fill reachable empty space using capsule collision, with `agent_height` and `agent_radius`, starting at `seed_pos`; keep solid voxels bordering reachable capsule space as final navigable/occlusion region, discard unreachable regions from final mesh.
>    - Config params: `voxel.size`, `voxel.opacity_threshold`, `voxel_fill.mode`, `voxel_fill.dilation_size`, `voxel_carve.agent_height`, `voxel_carve.agent_radius`, `voxel_carve.seed_pos`.
>    Verify closed room exterior-fill does not leak, open scene skips exterior-fill when seed connects to boundary, floor-fill works on terrain, capsule carve excludes too-narrow gaps.
> 15. Add occlusion/collision mesh export:
>    - mode `faces`: exposed-face extraction. For each solid voxel, emit quad only for faces adjacent to empty/out-of-grid cells. Cull faces between adjacent solid voxels. Output axis-aligned watertight voxel mesh. Use when collision/raycast must match voxel data exactly.
>    - mode `smooth` default: Marching Cubes over occupancy field to extract isosurface, then coplanar face merge to reduce triangle count and smooth voxel stair-steps.
>    - Output mesh from either mode is the only triangle input for step 16. It must come after voxelize -> fill -> carve, not raw voxelize.
>    Verify `faces` produces watertight cube with culled internal faces; verify `smooth` produces fewer triangles after coplanar merge; verify empty grid returns empty mesh safely.

>    - Config: agent height/radius, max slope, cell size, cell height, region settings.
>    - Output `navmesh.glb` and `navmesh.bin`.
